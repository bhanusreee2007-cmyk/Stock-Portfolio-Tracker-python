# Stock-Portfolio-Tracker-python

A simple and beginner-friendly **Stock Portfolio Tracker** built using Python that helps users calculate the total value of their stock investments. The program allows users to enter stock names and quantities, then computes the overall investment value using predefined stock prices stored in a dictionary.

This project provides a practical introduction to real-world financial tracking while helping beginners strengthen their understanding of core Python concepts such as dictionaries, user input handling, arithmetic operations, loops, and optional file handling.

## Features

* Allows users to enter stock names and quantities
* Uses a predefined dictionary to store stock prices
* Calculates the total value of the investment portfolio
* Provides a simple and interactive console-based interface
* Includes an option to save results in `.txt` or `.csv` format
* Beginner-friendly and easy to understand
* Demonstrates a real-world application of Python programming

##  Concepts Used

* Dictionaries
* User Input and Output
* Basic Arithmetic Operations
* Loops and Conditional Statements
* File Handling (Optional)
* Data Processing and Calculation

##  Objective

To understand how basic financial tracking systems work while enhancing Python programming skills and logical thinking through a practical, real-world inspired project. This project encourages learners to apply fundamental concepts to build a simple yet meaningful application.
